O projeto final consiste no desenvolvimento de uma aplicação web completa, uma lista de tarefas (To-Do List). A aplicação deve permitir que o usuário adicione, edite, remova e marque tarefas como concluídas, oferecendo uma experiência interativa e funcional. A persistência dos dados pode ser feita integrando-se a uma API ou utilizando o Local Storage.
 O usuário deve ser capaz de adicionar novas tarefas com título e estado inicial "pendente".
 As tarefas devem ser exibidas na tela ao carregar a página, mostrando o estado de cada uma.
 O usuário deve ser capaz de marcar/desmarcar tarefas como concluídas.
 O usuário deve ser capaz de remover tarefas da lista.
 A aplicação deve manipular o DOM dinamicamente.
 Utilizar eventos JavaScript (click, submit) para interação.
 Aplicar estilização simples com CSS e classes dinâmicas para estados das tarefas.
Utilizar Local Storage para armazenar tarefas localmente.
Opcionalmente, utilizar Fetch API para requisições HTTP (GET, POST, PUT, DELETE) para armazenamento e sincronização com um servidor. Estou disponibilizando uma API que simula um simples gerenciador de tarefa
